Setup
  Reference fit : MSGARCH 2.51 (Ardia 2019), Bayesian MCMC posterior-mean,
                  variance = sGARCH, distribution = norm.
                  Parameters loaded from results/msgarch_reference/models_K{K}.jld2.
  Filter        : Haas et al. (2004) path-independent MSGARCH state filter,
                  per-state conditional variance recursion
                    σ²_{t,k} = α0_k + α1_k R²_{t-1} + β_k σ²_{t-1,k}
                  with stationary initialisation σ²_{1,k} = α0_k / (1 - α1_k - β_k).
  Predictive    : f(R_t | F_{t-1}) = Σ_k P(s_t = k | F_{t-1}) · N(0, σ²_{t,k}).
                  VaR_α(t) = α-quantile of this Gaussian mixture, by binary
                  search on the mixture CDF.  Same Eq. (9) construction as the
                  body CHMM regime-conditional VaR (run_conditional_var.jl), with
                  CHMM state belief / Gaussian emissions replaced by MSGARCH ones.
  Tests         : Kupiec LR_uc (~chi²_1, 3.841)
                  Christoffersen LR_ind (~chi²_1, 3.841)
                  Christoffersen LR_cc  (~chi²_2, 5.991)
                  Engle-Manganelli DQ at q = 4 lags (~chi²_6, 12.59)

Reading. The body CHMM regime-conditional VaR (Table 6 in var_backtest.tex)
passes Christoffersen-cc cleanly at α = 0.05 on every (K, family) row, and
the higher-power Engle-Manganelli DQ test rejects at α = 0.01 only for the
K = 18 row. The MSGARCH panel above tests whether that conditional-coverage
behaviour is CHMM-specific or generic to multi-state regime-switching models
on this dataset, by reusing the same Eq. (9) state-filter pipeline on MSGARCH
state forecasts. Side-by-side reading: any row where MSGARCH passes Kupiec but
rejects Christoffersen-ind / -cc indicates that breach clustering is present in
the MSGARCH regime-conditional construction too, isolating the conditional-
coverage benefit to the multi-state structure rather than to CHMM specifically.
